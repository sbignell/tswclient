Sid likes wine and Sven likes design. They are your Node.js server and single-page app.

Vision

To provide a simple, minimal starter project for creating single-page apps built with node.js on the server, backbone on the client and a MySQL data store. This project was created based on jedireza's fantastic Drywall project and Christophe Coenraets' Wine Cellar tutorial.

Technology

Sid and Sven utilise current technologies such as node.js, express, passport, mysql ,sequelize on the back end and backbone, bootstrap, font awesome, jade and less on the front end. They're built using Grunt.

Open

Open source and available on GitHub, go ahead and fork them. I hope you build something great!
